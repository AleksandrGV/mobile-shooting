Программа курса

1

Настройка камеры телефона

Изучите все возможности камеры вашего смартфона. Научитесь правильно настраивать параметры для разных условий съемки.

2

Настройка камеры через приложения

Откройте для себя профессиональные приложения для съемки, которые расширят возможности вашего смартфона.

3

Как снимать себя дома, креативные ракурсы

Научитесь создавать качественный контент в домашних условиях. Узнайте секреты креативных ракурсов и композиции.

4

Монтаж на телефоне

Освойте мобильные приложения для монтажа. Научитесь создавать профессиональные видео прямо на своем смартфоне.

5

Советы по видам съемок

Получите практические советы по различным видам съемок: интервью, влоги, продуктовая съемка, сторис и многое другое.

Дополнительные возможности курса

  • Общий чат - чат единомышленников с дополнительной информацией, обратной связью и проверкой домашних заданий
  • Созвоны - разбор домашних заданий и ваших запросов
  • Бонусы - множество полезных материалов, которые будут раскрываться постепенно
